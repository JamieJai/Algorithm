# 구름 (LEVEL.2) 수열
## part. 구현

### 문제
---
F(n)이 어떤 수열의 n번째 값이라고 하면, 다음과 같은 공식이 성립한다.
- F(1) = 0, F(2) = 1
- F(n) = F(n-1) + F(n-2),(n > 2)

이때 K가 주어지면, F(K) 값을 출력하시오.

### 입력
---

첫째 줄에 K가 주어진다.
• 1≤ K ≤ 100 000

### 출력
---

F(K)을 1 000 000 007로 나눈 나머지를 출력하시오.

### 예시 1
#### 입력
```
6
```
#### 출력
```
5
```

### 예시 2
#### 입력
```
10
```
#### 출력
```
34
```
